# Garden Calm - Инструкции по разработке

## 🚨 КРИТИЧЕСКИЕ ПРАВИЛА РАЗРАБОТКИ

### 1. GitHub Синхронизация
- ПОСЛЕ КАЖДОГО БОЛЬШОГО ЭТАПА (не реже чем каждые 30 минут работы) - ОБЯЗАТЕЛЬНО:
  Обновлять GitHub репозиторий и заливать файлы туда
- БЕЗ GITHUB КОММИТОВ = ПОТЕРЯ ВСЕЙ РАБОТЫ

### 2. Дизайн чата - ПРИОРИТЕТ #1
- Чат должен быть ВИЗУАЛЬНО ПОТРЯСАЮЩИМ
- Вдохновение: Claude.ai от Anthropic (минимализм + элегантность)
- Обязательные элементы:
  * Плавные анимации сообщений (Framer Motion)
  * Красивый градиентный фон (от #10B981 к белому)
  * Эффект "печатания" для ответов AI
  * Мягкие тени и современные скругления
  * Адаптивный дизайн (особенно для мобильных)
  * Микроанимации при наведении
  * Элегантные переходы между состояниями

### 3. Структура работы
1. Читать этот файл в начале КАЖДОЙ сессии
2. Проверять README.md для понимания прогресса
3. Продолжать с последнего незавершенного этапа
4. Обновлять прогресс в README.md

### 4. UI/UX Принципы
- "Первое впечатление решает всё" - чат должен WOW с первого взгляда
- Каждый пиксель имеет значение
- Если сомневаешься - выбирай более красивый вариант
- Тестируй на разных экранах

### 5. Технические требования красоты
- Tailwind CSS с кастомными анимациями
- CSS переменные для плавных переходов темы
- Оптимизированные SVG для иконок
- Lazy loading для изображений медитаций
- 60 FPS анимации (использовать transform, не position)
